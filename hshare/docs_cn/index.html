HShare是一个简单的社交分享JQuery插件，用户通过编写简单的配置项即可将分享按钮嵌入至网页中。 将本插件直接部署在支持HTTPS的网页中即可让分享接口支持HTTPS。

安装

bower

bower install --save bower

HTML

<link rel="stylesheet" href="hshare.min.css">
<script type="text/javascript" href="jquery.min.js"></script>
<script type="text/javascript" href="hshare.min.js"></script>

本插件依赖于jQuery，因此在使用本插件之前请确保jQuery已经载入。

基本使用

步骤1：在HTML中指定容器：

<html>
    <head>
        <!-- something here -->
    </head>
    <body>
        <div id="container"></div>
        <!-- something here -->
    </body>
</html>

步骤二：在JS中按如下方式配置HShare

$("#container").hshare();

hshare方法还可接收用户自定义的配置项：

$("#container").hshare({
    size: "large",
    copyLink: false,
    platforms: [{
        name: "wechat"
    }, {
        name: "qzone"
    }]
});
